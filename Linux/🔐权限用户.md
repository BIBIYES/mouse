# 权限用户

## 查询用户和组

1. 查询用户

    ```bash
    getent passwd
    ```

2. 查询组

    ```bash
    cut -d: -f1 /etc/group
    ```

## 操作用户

1. 添加用户

    添加一个用户生成用户文件夹

    ```sh
    useradd -m mouse
    ```

2. 删除用户

    删除一个用户

    ```bash
    userdel mouse
    ```

## 添加用户到组

1. 将用户添加到 `sudo`​ 组

​`sudo`​ 组允许用户执行管理员命令。要将用户添加到 `sudo`​ 组，可以使用以下命令：

```
sudo usermod -aG sudo 用户名
```

* ​`sudo`​：以管理员权限执行命令。
* ​`usermod`​：修改用户账户的命令。
* ​`-aG`​：将用户追加到指定的组中（`-a`​ 表示追加，`-G`​ 表示组）。
* ​`sudo`​：目标组名。
* ​`用户名`​：要添加到组的用户名。
